# When the Machine Stops Looking

Research project on autonomous replenishment, censored demand, and the value of a human filter in inventory management.

## Contents

- `autonomous_replenishment_intro.tex` — main paper draft
- `references.bib` — bibliography
- `fold_check.py` — fold-check utility
- `MECHANISM_RECONSTRUCTION.md` — mechanism reconstruction notes

## Build

Compile the paper with `latexmk`:

```bash
latexmk -pdf autonomous_replenishment_intro.tex
```
